Detailed Description

Returns the absolute value of the center of element 0.

For a convergence residual (e.g. de = e - e_old) this is the quantity the convergence check bounds by the tolerance – the incomplete-convergence error – NOT the propagated uncertainty (which is the radius and is of the same order as the energy's own uncertainty). Dispatched on the element type; for non-UQ types the center is just the value.
